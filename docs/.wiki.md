# 项目概要

API 测试用例生成代理（API Test Case Generation Agent）旨在基于 OpenAPI 规范自动化生成 API 测试用例。本项目利用先进的 AI 模型和结构化工作流来解析 OpenAPI 文档，生成全面的测试计划，并生成多种编程语言的可执行测试代码。它的目标是提高 API 测试的效率和准确性，使其成为开发人员和 QA 工程师的必备工具。

# 项目模块描述

*   **OpenAPI 解析 (OpenAPI Parsing)**：解析 OpenAPI 文档以提取相关信息。
*   **LangGraph 工作流 (LangGraph Workflow)**：管理通过定义节点（解析器、规划器、生成器）的操作流。
*   **测试用例生成 (Test Case Generation)**：基于解析后的 OpenAPI 规范生成测试用例，支持多种编程语言 (cURL, Java, Go)。
*   **多模型适应性 (Multi-model Adaptability)**：根据不同 AI 模型的能力优化性能。

# 文件描述清单

*   **PRD.md**：包含产品需求详情。
*   **architect.plantuml**：展示系统架构的图表。
*   **class_diagram.plantuml**：类结构和关系。
*   **sequence_diagram.plantuml**：操作的工作流序列。
*   **system_design.md**：实施的详细设计规范。
*   **todo.md**：开发任务及其完成状态。
*   **verify_setup.py**：验证设置和导入的脚本。
*   **debug_import.py**：用于导入问题的调试脚本。

# 技术栈

*   **前端**：React, TypeScript, Vite, Shadcn-ui, Tailwind CSS
*   **后端**：Python 3.10+, FastAPI
*   **代理编排**：LangGraph
*   **数据解析**：PyYAML, Pydantic
*   **LLM 集成**：LangChain

# 用法

1.  安装依赖项：
    ```bash
    pip install -r backend/requirements.txt
    ```
2.  运行 FastAPI 服务器：
    ```bash
    uvicorn backend.main:app --reload
    ```
